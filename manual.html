1. 基本的な使い方

Easy Ecomap JPは、ブラウザ上で直感的にエコマップを作成できる無料ツールです。

エコマップとは？

エコマップ（Ecomap）は、個人や家族を取り巻く社会環境や人間関係を図式化したものです。 ソーシャルワークやカウンセリング分野で広く使用されており、関係性を視覚的に理解するのに役立ちます。

  1. ブラウザでページを開く
  2. ツールバーから必要なツールを選択
  3. キャンバス上でクリック・ドラッグして要素を配置
  4. 要素をダブルクリックしてテキストを編集
  5. 完成したら保存ボタンでファイルに保存

2. ツールの使い方

選択ツール

要素を選択・移動・サイズ変更する基本ツール

  • • クリック：要素を選択
  • • ドラッグ：要素を移動
  • • 四隅をドラッグ：サイズ変更
  • • 複数選択：ドラッグで範囲選択

項目追加ツール

新しい項目（四角形）を追加

  • • クリック：項目を配置
  • • ドラッグ：サイズを指定して配置
  • • ダブルクリック：テキスト編集

テキストツール

自由なテキストを追加

  • • クリック：テキストを配置
  • • ダブルクリック：編集モード
  • • Enter：編集完了

消しゴムツール

不要な要素を削除

  • • クリック：要素を削除
  • • 複数削除：選択後にDeleteキー

3. 図形の操作

図形の移動

  1. 選択ツールをクリック
  2. 移動したい図形をクリックして選択
  3. 図形をドラッグして目的の位置に移動

図形のサイズ変更

  1. 図形を選択
  2. 四隅または辺に表示されるハンドルをドラッグ
  3. 最小サイズは自動的に制限されます

図形のテキスト編集

  1. 図形をダブルクリック
  2. テキストを編集
  3. Enterキーまたは他の場所をクリックで完了

4. 線の描画

線の種類

強い・肯定的な関係
普通の関係
希薄・疎遠な関係
ストレス・葛藤関係

線の描画方法

  1. 線の種類を選択（強い・普通・希薄・ストレス）
  2. 開始点から終了点までドラッグ
  3. 図形の中心に近づけると自動的にスナップ
  4. マウスを離すと線が完成

線の編集

  1. 線を選択
  2. 両端のハンドルをドラッグして位置調整
  3. 線を移動して全体の位置を変更

5. テキストの追加・編集

テキストの追加

  1. テキストツールを選択
  2. キャンバス上の任意の場所をクリック
  3. テキストを入力
  4. Enterキーで確定

テキストの編集

  1. テキストをダブルクリック
  2. 編集モードでテキストを修正
  3. Enterキーまたは他の場所をクリックで完了

文字サイズの変更

  1. テキストまたは図形を選択
  2. ツールバーの文字サイズボタンをクリック
  3. 小・中・大から選択

6. 保存・読み込み

JSON形式での保存（推奨）

  • 後から編集可能
  • データの完全性が保たれる
  • 図形、線、テキストのすべての情報を保持
  • ファイルサイズが小さい

PNG形式での保存

  • 画像として保存
  • 印刷やプレゼンテーションに適している
  • 余白の調整が可能
  • 背景の透過設定が可能

ファイルの読み込み

  1. 「読込」ボタンをクリック
  2. 保存したJSONファイルを選択
  3. 自動的にエコマップが復元される

7. キーボードショートカット

基本操作

元に戻す Ctrl+Z
やり直す Ctrl+Y
削除 Delete
キャンセル Escape

編集操作

テキスト編集完了 Enter
改行（テキスト編集中） Shift+Enter

8. 便利な機能とコツ

グリッド表示

ヘッダーの「グリッド」チェックボックスで、背景にグリッドを表示できます。 要素の配置を整列させるのに便利です。

自動スナップ機能

線を描画する際、図形の中心に近づけると自動的にスナップします。 正確な接続が簡単にできます。

複数選択

選択ツールでドラッグして範囲を指定すると、複数の要素を一度に選択できます。 選択後は一括で移動や削除が可能です。

履歴機能

すべての操作は履歴に保存されます。「元に戻す」「やり直す」ボタンで 任意の時点に戻ることができます。
